== Description ==

TweetBottom quickly and easily assist you with growing your Twitter following by adding a follow me button after each post you write. You can choose the Twitter account you want to use as well as the colours for the follow me button.

If you have any comments or questions, please visit my site @ [http://www.imod.co.za/wordpress-plugins](http://www.imod.co.za/wordpress-plugins)

== Installation ==

Follow the steps below to install the plugin.

1. Upload the `TweetBottom` folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to TweetBottom settings to edit your settings.
